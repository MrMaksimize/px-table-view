Table Row Controls

Table controls provide information, indicate a state, or present actions for table row items. Table controls appear as icons to the left or right of row text.

Table control placement:

  • States and primary actions are placed on the left side of a list row.
  • Secondary actions and info are placed on the right side of the row.

Types

  • Checkbox
  • Switch
  • Reorder
  • Expand/collapse

Checkbox

A checkbox can be either a primary or secondary action.
