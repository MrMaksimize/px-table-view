    Table View Modifiers

    flush Enables no padding
    tiny Enables tiny padding
    small Enables small padding
    large Enables large padding
    huge Enables huge padding
    inset Enables margin and border
    bare Removes all borders / background colors

    Table Row Specs

    A table-view consists of a single continuous column of tessellated sub-divisions of equal width called rows that function as containers for tiles. Tiles hold content, and can vary in height within a table-view.
